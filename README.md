# 🚀 Curso Django Web Framework & DRF

Repositório dedicado ao meu aprendizado prático com **Django Web Framework** e **Django Rest Framework (DRF)**.

Aqui estou documentando minha evolução no desenvolvimento web com Python, focando principalmente no **back-end com Django**, explorando desde fundamentos até conceitos avançados.

---

## 🧠 Sobre o Projeto

Este repositório faz parte do meu processo de aprendizado em desenvolvimento web com Django, onde aplico na prática conceitos como:

* Estrutura de projetos Django
* Arquitetura MTV (Model, Template, View)
* ORM (Object Relational Mapper)
* Criação de APIs com DRF
* Autenticação de usuários
* Testes automatizados
* Deploy de aplicações

## ⚡ Como executar o projeto

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/seu-repo.git](https://github.com/Monaliza-Vasconcelos/curso-django-projeto1.git

# Entrar na pasta
cd curso-django-projeto1

# Criar ambiente virtual
python -m venv venv

# Ativar ambiente
# Windows
venv\Scripts\activate
# Linux / Mac
source venv/bin/activate

# Instalar dependências
pip install -r requirements.txt

# Rodar o servidor
python manage.py runserver
```
